8. Comandos ejecutados y resultado observado:

Comando: whoami
Resultado: para ver el nombre del usuario

Comando: hostname
Resultado:para ver el nombre del servidor

Comando: date
Resultado:para ver fecha y hora

Comando: pwd
Resultado:para ver la ruta de la carpeta

Comando: ls
Resultado:para ver la lista de archivos

Comando: ls -la
Resultado: sirve para mostrar todod los archivos contantdo los ocultos

Comando: mkdir practica-linux
Resultado:para crear carpetas

Comando: cd practica-linux
Resultado:para entrar a la carpeta

Comando: touch notas.txt
Resultado:para crear el archivo de notas.txt

Comando: echo "Mi primera practica en terminal Debian" > notas.txt
Resultado:crea una carpeta notas.txt y dentro de el archivo aparece lo que esta dentro de las comillas

Comando: cat notas.txt
Resultado:para ver el contenido del archivo

Comando: cp notas.txt copia-notas.txt
Resultado:para copiar lo que esta en notas.txt y pegarlo en copia-.txt

Comando: mv copia-notas.txt notas-renombradas.txt
Resultado:sirve para modificar el nombre del archivo

Comando: mkdir respaldo
Resultado:crea la carpeta respaldo

Comando: mv notas-renombradas.txt respaldo/
Resultado:cambia el nombre de respaldo a notas-renombradas.txt

Comando: df -h
Resultado:sirve para ver el espacio disponible y el espacio utilizado del disco duro

Comando: free -h
Resultado:para ver cuanta memoria ram tiene disponible y cuanta esta utilizando en tiempo real

Comando: uname -a
Resultado:sirve para ver la información tecnica del sistema operativo y el nucleo

Comando: history
Resultado:para ver todos los comando utilizados

Comando: ip a
Resultado:muestra todas las interfaces

Comando: top
Resultado:para mirar los procesos o tareas en tiempo real
